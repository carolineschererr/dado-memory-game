Gráficos

Bar chart / Gráfico de barra

Um gráfico de barras é um gráfico com barras retangulares cujos comprimentos são proporcionais aos valores representados. As barras podem ser verticais ou horizontais. Um eixo do gráfico mostra as categorias específicas que estão sendo comparadas enquanto o outro eixo representa um valor discreto.

Os gráficos de barras fornecem uma apresentação visual de dados categóricos. Dados categóricos são um agrupamento de dados em grupos discretos, como meses do ano, faixa etária, tamanhos de calçados e animais. Essas categorias são geralmente qualitativas. As barras no gráfico podem ser organizadas em qualquer ordem.

Boxplot

Em estatística descritiva, um boxplot é uma maneira conveniente de representar graficamente grupos de dados numéricos por meio de seus quartis. Um box plot exibe mediana, quartis superior/inferior e máximo/mínimo. Outliers podem ser plotados como pontos individuais. Os espaçamentos entre as diferentes partes da caixa indicam o grau de dispersão (espalhamento) e assimetria dos dados e mostram valores discrepantes. Box plots podem ser desenhados horizontalmente ou verticalmente.

O gráfico de violino é semelhante aos gráficos de caixa, exceto que eles também mostram a densidade de probabilidade dos dados em valores diferentes.

Hanging rootgram

A comparação da distribuição de dados com uma distribuição teórica de um histograma comum pode ser difícil porque as frequências pequenas são dominadas pelas frequências maiores e é difícil perceber o padrão de diferenças entre as barras do histograma e a curva. Portanto, John Tukey introduziu o Rootograma suspenso em 1971 (também chamado de Rootograma suspenso de Tukey) para resolver esses problemas. Nesta visualização, a comparação é facilitada ao "pendurar" os resultados observados da curva teórica, de modo que as discrepâncias sejam vistas por comparação com o eixo horizontal em vez de uma curva inclinada. Como no rootograma, o eixo vertical é dimensionado para a raiz quadrada das frequências, de modo a chamar a atenção para as discrepâncias nas caudas da distribuição.

É uma variação do conceito de histogramas e gráficos de Pareto combinando observado e previsto de uma forma simples onde os gráficos de linhas mostram que os dados estão mudando continuamente.

Gráfico de áreas aninhadas

Um gráfico proporcional de áreas aninhadas é usado para comparar proporções (tamanho, quantidades, etc.) para fornecer uma visão geral rápida do tamanho relativo dos dados sem o uso de escalas. Você também pode encontrar uma variante disso com formas em formato de círculo.

Visualização de rede

A visualização de rede (também chamada de gráfico de rede) é frequentemente usada para visualizar relacionamentos complexos entre uma grande quantidade de elementos. Uma visualização de rede exibe estruturas gráficas não direcionadas e direcionadas. Este tipo de visualização ilumina as relações entre as entidades. As entidades são exibidas como nós arredondados e as linhas mostram as relações entre elas. A exibição vívida dos nós da rede pode destacar discrepâncias de dados não triviais que, de outra forma, passariam despercebidas.

Diagrama de acordes sem fita

Um diagrama de acordes sem fita é uma versão simplificada de um diagrama de acordes, com apenas as linhas de conexão sendo exibidas. Isso fornece mais ênfase nas conexões dentro dos dados. Um diagrama de acordes é um método gráfico de exibir as inter-relações entre os dados em uma matriz. Os dados são organizados radialmente em torno de um círculo com as relações entre os pontos.

Gráfico de pizza

Um gráfico de pizza é dividido em setores, ilustrando a proporção numérica. Em um gráfico de pizza, o comprimento do arco de cada setor (e conseqüentemente seu ângulo central e área) é proporcional à quantidade que ele representa. Embora seja nomeado por sua semelhança com uma torta fatiada, há variações na forma como pode ser apresentado.

Gráfico de pizza sobre mapa

Um mapa de gráfico de pizza é simplesmente uma combinação de uma visualização de dados de gráfico de pizza e um mapa. É usado para visualizar localização e proporção numérica de maneira simples. Às vezes, você verá uma combinação de gráfico de pizza, mapa e gráfico de bolhas. Onde o tamanho do círculo do gráfico de pizza permite mais uma dimensão para a visualização.

Pirâmide

Uma pirâmide populacional, também chamada de pirâmide de idade ou diagrama de imagem de idade, é uma ilustração gráfica que mostra a distribuição de vários grupos de idade em uma população, que forma a forma de uma pirâmide quando a população está crescendo. Também é usado em ecologia para determinar a distribuição etária geral de uma população; uma indicação das capacidades reprodutivas e probabilidade de continuação de uma espécie.

Gráfico de barra radial

Um Gráfico de barras radiais circulares refere-se simplesmente a um gráfico de barras típico exibido em um sistema de coordenadas polares, em vez de um sistema cartesiano.

Ele é usado para mostrar comparações entre categorias usando uma forma circular.

Gráfico de linha radial

Gráfico de linha radial consiste em uma parte dos gráficos radiais que pega os dados e os renderiza como uma coleção de pontos de dados enrolados em um círculo. Também está mapeando uma lista de categorias do mínimo ao máximo da extensão do gráfico. Radial Line Graph é renderizado usando uma coleção de linhas retas conectando pontos de dados.

Gráfico de áreas empilhadas

O gráfico de área empilhado é semelhante ao gráfico de área simples, mas aqui usa várias séries de dados que iniciam cada ponto a partir do ponto deixado pela série anterior. É útil para comparar várias variáveis que mudam ao longo do intervalo.

Treemap

Os Treemaps exibem dados hierárquicos (estruturados em árvore) como um conjunto de retângulos aninhados. Cada ramo da árvore recebe um retângulo, que é então lado a lado com retângulos menores representando sub-ramos. O retângulo de um nó folha tem uma área proporcional a uma dimensão especificada nos dados. Freqüentemente, os nós de folha são coloridos para mostrar uma dimensão separada dos dados.

Nuvem de palavras

Uma nuvem de tags é uma representação visual para dados de texto, normalmente usada para representar metadados de palavras-chave (tags) em sites, para visualizar texto de forma livre ou para analisar discursos (por exemplo, campanha eleitoral). As tags geralmente são palavras únicas e a importância de cada tag é mostrada com tamanho ou cor da fonte. Este formato é útil para perceber rapidamente os termos mais proeminentes e para localizar um termo em ordem alfabética para determinar sua proeminência relativa.
